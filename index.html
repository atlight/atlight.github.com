Keyboard shortcut table

For the Common Good has keyboard shortcuts (really access keys) for most elements. You can view them by pressing Alt, or refer to the list below:

Top of window

Alt+NName text box
Alt+GGo button
Alt+RRandom file/Next file from list button
Alt+OOther source button
Alt+SSettings button

Middle of window

Alt+PPage on English Wikipedia text area
Alt+KPage on Wikimedia Commons text area

Bottom of window

Alt+VSelect version button
Alt+MView metadata button
Alt+FNew filename on Commons text box
Alt+WIgnore warnings check box
Alt+ATag/delete local file ... check box
Alt+TTransfer button
Alt+EView ... on English Wikipedia hyperlink
Alt+CView ... on Wikimedia Commons hyperlink
