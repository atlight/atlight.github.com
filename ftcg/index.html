For the Common Good

In 2011, I put out the first release of For the Common Good, a tool to allow Wikimedia users to transfer images and media files from their local wiki to Wikimedia Commons, the central repository. FtCG has been largely superseded by other tools, including FileImporter, and is no longer maintained. However, it still works and remains available for download.

Find out more about For the Common Good at User:This, that and the other/For the Common Good on English Wikipedia.

You can still download the final release, For the Common Good 1.2 (released 23 February 2020). For system requirements and so forth, see the English Wikipedia page.

Not on English Wikipedia? See the list of local wiki data files, or write your own.

Don't speak English? Download a language file to use FtCG in your own language.

If you're a coder, you can download or browse the source code on GitHub.
